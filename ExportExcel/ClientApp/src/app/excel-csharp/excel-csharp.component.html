Gerando o excel atráves do C#

O C# obtem os dados da base ou de qquer outro lugar e converte esses dados num datatable ou dataset, e usando a biblioteca XCloseXML você cria o excel (colunas, linhas, cores de linhas, bordas e etc, etc, etc), usa o datatable ou dataset para os dados das linhas. O endpoint da API retorna o stream desse excel para o angular se for para um blob ou uma string do tipo base64.
